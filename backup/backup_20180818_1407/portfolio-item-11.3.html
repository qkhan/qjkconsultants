Project Description


Designed, programmed and implemented a completely new approach to the vision system. Employing third party neural networks libraries to recognise any pattern regardless of minor distortion to it. The vision system has the capability to also add new patterns for recognition. The system was to interact with the rest of the components such as the Loader Robot, conveyor, Assembler Robot, Petra and SCADA to produce an autonomous system for a chocolate factory.
