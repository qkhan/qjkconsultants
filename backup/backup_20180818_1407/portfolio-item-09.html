Project Description


Developed an Online Stock Management Website for Allianz insurance to add, update or remove stocks, upload pde’s for electronic delivery files, automate stock activation facility and management of master control sheet by customer service department.

Solution Overview


  1. Acxiom Log Monitoring Web site provides client services/Implementation team a Graphical User Interface to keep track of the running scheduled jobs for various active projects, using local internet browser

  2. The web site provides current status of running jobs, with the time and date stamp for each run.

  3. It allows user to visually track projects with longer processing duration or if the project has failed or successfully completed

  4. The user could drill down a project history log, and has access to the log files via web. One could select a project and check the previous run logs.

  5. The history log also provide statistics related to each run, which is useful for comparing various project runs, especially if the processing has failed due to some addition of the code, not previously present in the older runs

  6. Through the summary page, all the other logs related to the project can be accessed.

  7. The user could save the log files to their personal network drives, etc. However, for security reason, users are only allowed to read the log file. Once save to their network drive, additional comments can be added to the personal copy of the log file.


Project Technical Detail


The web site is developed in ASP.net with C# as a back end programming language.
